# Case Study: Botium Toys Security Audit

## Objective
Conducted a comprehensive security audit for Botium Toys to assess their current security posture, ensure compliance with international standards (GDPR, PCI DSS, SOC), and mitigate risks to their digital assets.

## Scope & Frameworks
* **Standards:** PCI DSS (Payment Card Industry), GDPR (Data Privacy), SOC (Service Organization Control).
* **Controls:** Evaluated Administrative, Technical, and Physical security controls.
* **Risk Assessment:** Utilized a 1-10 risk scoring matrix to prioritize remediation efforts.

## Key Findings
* Identified gaps in legacy system encryption.
* Analyzed access control vulnerabilities (Principle of Least Privilege).
* Assessed physical security measures for on-site servers.

## Results
Delivered a prioritized remediation roadmap with a 12-week implementation timeline and a detailed cost-benefit analysis for the executive board. **Initial Risk Score: 8/10.**
